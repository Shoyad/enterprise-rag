Visual proof that the system works — technical clients often look here before the code.

- `architecture.png` — rendered architecture diagram
- `screenshots/` — API responses with citations, ingestion status
- `sample-requests.http` — copy-paste `curl` / REST-client examples
- `demo.mp4` — 20–30s screen recording: ingest → ask → cited answer

Everything shown here uses the generated, fictional demo knowledge base. No real client
data appears in any asset.
